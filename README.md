# ColorMe

カラーミーのAPI（ http://shop-pro.jp/?mode=api_started ）をRubyから扱うためのラッパーgem

## Installation

Add this line to your application's Gemfile:

    gem 'color_me', github: 's-osa/color_me'

And then execute:

    $ bundle
